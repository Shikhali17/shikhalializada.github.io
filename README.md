# Shikhali Alizada - Academic Website

Personal academic website for Shikhali Alizada, Finance Professor.

## Features

- **Home Page**: Introduction with research focus areas
- **Bio Section**: Educational background and professional summary
- **Research Section**: Research interests, working papers, and publications
- **Teaching Section**: Current courses and teaching philosophy
- **Contact Section**: Contact information and office details
- **CV Page**: Comprehensive curriculum vitae

## Design

- Clean, professional academic design
- Responsive layout (works on desktop, tablet, and mobile devices)
- Smooth scrolling navigation
- Print-optimized CV page

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript
- No frameworks or build tools required

## Local Development

To view the website locally:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Customization

To customize the website for your own use:

1. Update personal information in `index.html` and `cv.html`
2. Replace placeholder text with your actual content
3. Add your publications, research papers, and teaching experience
4. Update contact information
5. Customize colors in `styles.css` if desired
